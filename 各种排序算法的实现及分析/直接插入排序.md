# 直接插入排序

标签（空格分隔）： 排序

```java
public void insertSort(int a){
    if(a!=null){
        for(int i=1;i<a.length;i++){
            int temp=a[i],j=i;
            if(a[j-1]>temp){
                while(j>=1&&a[j-1]>temp){
                a[j]=a[j-1];
                j--;
                }
            }
            a[j]=temp;
        }
    }
}
```
