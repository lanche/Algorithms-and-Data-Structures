# 冒泡排序

标签（空格分隔）： 排序

---

```java
void BubbleSort(int[] array){
    length=array.length;
     for(int i=0;i<length-1;i++){                       //n个元素只要比较找n-1个元素，故i<length-1
        for(int j=length-1;j>i;j--){
             if(array[j-1]>array[j]){
                int temp = array[j];
                    array[j]=array[j-1];
                    array[j-1]=temp;
            }
            
         }
}
```
